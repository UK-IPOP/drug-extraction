Important Security Notes: It is import to note that this server is running locally on your machine and thus, if your machine meets the data security standards for your project, then this interface is acceptable to use, just like the command line interface. All processing is done on the host device.

The expected file type is `*.csv` and the program will not accept other formats.

Clean runs remove the existing output files from previous runs, toggle this off to append to previous records.

Strict restricts matches to only exact matches (similarity=1.0).
